1.1 访问子串

问题

想知道一个字符串是否包含一个特定的子串。例如，想查看一个 email 地址是否包含一个 @

实现

strpos 找到子串时，会返回子串的位置。找不到子串时，会返回 false，为了区分 0 和 false，必须使用 恒等操作符或非恒等操作符来进行判断。
